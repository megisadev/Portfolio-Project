# Portfolio-Project
Portfolio project for Chegg. 

Pete's artist portfolio web page - a project for Web Development Chegg Program.  A simple one-page website that briefly showcases Pete's artwork as well as some personal information and background. 

No AI tools were used. Followed design instructions clearly to create a page inspired by the original design mock-up. 
Learned a bit more about structuring a web page after confusing myself a bit.  Validating the HTML was a huge help in learning what areas I needed help in.
